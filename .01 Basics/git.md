# Git commands

## Clone repository
```git
git clone [URL]
```

## Status of files in working directory
```git
git status
```

## Stage changes
```git
# Specific file
git add [FILE] 
# current directory
git add . 
```

## Commit to repository
```git
git comming -m [MESSAGE]
```

## Difference of unstaged and working directory
```git
# unstaged
git diff
# changes moved to staging
git diff --staged
```
